# split

## Description

Split a tab between friends by quickly uploading a photo of a receipt and easily Venmo requesting friends for their portion of the bill.

### Frontend

- Runs on localhost:3000

#### Dependencies

- Next.js
- axios
- Sass

#### How to Run

- next-auth requires 16.13.0. I recommend installing different versions of node with nvm.
- $ npm install
- $ npm run dev

### Backend

- Runs on localhost:8080

#### Dependencies

- morgan
- body-parser
- nodemon
- express
- cors
- mongoose
- dotenv
- @veryfi/veryfi-sdk
- multer
- bcrypt
- jsonwebtoken

#### How to Run

- $npm i -g nodemon
- $ nodemon
